# DnD5e - System Customizer

<ModuleInfo/>

Customize the D&D5e system to your liking.

## How to use

Simply head to the `Configure Settings` menu, then head to the 'DnD5e - System Customizer' customization menu.

## Limitations

None currently known.

## Incompatibilities

### Token Magic FX

There is currently an unknown incompatibility with the Token Magic FX module. When the module is enabled you will be unable to save/edit your settings.

There is a simple workaround for this issue. Simply disable the Token Magic FX module, save your settings, then re-enable the Token Magic FX module.

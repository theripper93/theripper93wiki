# Pixelize Dice

<ModuleInfo/>

A set of animated dice for Dice So Nice! These dice are exclusive to Early Access Patreons.

## How to use

- Enable the module
- Open you Dice so Nice settings
- Select one of the `Wave` dice options
- Use the `Wave - White` then set a color for custom colors
